# LIFE GRID

Карта вашей жизни по неделям в стиле GitHub-контрибьюшнов. Время течёт **сверху вниз**: каждая строка — год жизни, каждая ячейка — одна неделя. Отмечайте, что делали в каждый день — чем больше выполненных задач, тем ярче горит день, а вместе с ним и неделя.

![preview](docs/preview.png)

## Возможности

- **Сетка жизни** — недели вашей жизни, расположенные сверху вниз (по правилам классической «life in weeks»). Количество недель считается из даты рождения и ожидаемой продолжительности жизни.
- **Клик по неделе → 7 дней** — выбираете день внутри недели и записываете, что делали.
- **Яркость по задачам** — 5 уровней насыщенности зелёного (как на GitHub): чем больше выполненных задач за день/неделю, тем ярче ячейка.
- **Статистика** — прожито дней, недель, возраст, процент прожитой жизни, прогресс-бар.
- **Подсказки** — наведите на ячейку, чтобы увидеть номер недели, диапазон дат и число задач.
- **Локальное хранение** — все данные лежат в `localStorage` браузера. Есть экспорт/импорт JSON.

## Как считается яркость

| Уровень | День (выполнено задач) | Неделя (сумма за 7 дней) |
| ------- | ---------------------- | ------------------------ |
| 0       | 0                      | 0                        |
| 1       | 1–2                    | 1–4                      |
| 2       | 3–4                    | 5–9                      |
| 3       | 5–6                    | 10–16                    |
| 4       | 7+                     | 17+                      |

## Запуск

Это статический сайт без сборки и бэкенда. Достаточно открыть `index.html`:

```bash
# любой статический сервер, например:
npx http-server . -p 8080
# затем откройте http://localhost:8080
```

Или просто откройте файл `index.html` в браузере.

## Первый запуск

При первом открытии откроется окно настроек — укажите **дату рождения** и **ожидаемую продолжительность жизни**. После этого сетка привяжется к вашей реальной жизни: текущая неделя будет обведена, прошедшие недели активны, будущие — приглушены.

## Стек

Чистые HTML + CSS + JavaScript, без зависимостей.

- `index.html` — разметка
- `styles.css` — тема в стиле GitHub (dark)
- `app.js` — логика сетки, дней, задач и хранения
